LISTAS DENTRO DO HTML5
ul = NÃO ORDENADA
<nav>
	<ul>
			<li>Item da lista</li>
	</ul>
</nav>
ol = ORDENADA (númera os itens)
<nav>
	<ol>
		<li>Item da lista</li>
	</ol>
</nav>
dl = DESCRITIVA
<nav>
	<dl>
		<dt>Item da lista</dt>
			<dd>Descrição do item</dd>
	</dl>
</nav>

TAGS DE TABELA
<tr></tr> linha da tabela
<td></td> tag de célula
<th></th> cabeçalho tabela na area de celula
<table></table> define a tablea e border da borda
// número de celulas tem de ser igual nas linhas pra não desconfigurar
        <table border>

                <td>

                </td>
                <td>

                </td>
            </tr>
        </table>

TAG SEMANTICA DE TABELA
<thead></thead> Define o cabeçalho
<tbody></tbody> Define o corpo da tabela
<tfoot></tfoot> Define o roda pé da tabela
<caption></caption> Definição do título da tablea

<table>
    <caption>Título tabela para busca</caption>
    <thead>

    </tfoot>

LISTAS DENTRO DO HTML5
<form></form> Classificação para form
action="Banco" para onde os dados vão ser enviados
<label></label> rotulo da parte seção com o input
<input> a entrada do dado
type="text" identificação do tipo da entrada como de texto
name="nome" nomeação do campo
type="submit" Botão de envio
type="number" Tipo número
type="date" Tipo data
type="email" Tipo e-mail
placeholder="email@xyz.com" Placeholder pra se o campo estiver vazio
type="password" Tipo senha
required patter="[A-Z]{3}[0-9]{4}" Dentro do colchetes os limites e dentro das chaves o mínimo
tittle="texto" Cria um título para o input
<fieldset></fieldset> Define a área de um campo

<form action="/page.php" method="post">
    <label for="fname">Nome:</label>
    <input type="text" id="fname" name="nome">
    <input type="submit" value="Submit">
</form>
